## Usage

With argument:

``topsy-turvy "miao"``

or

``tt "miao"``

With stdin:

```echo "miao" | topsy-turvy ```

or

```echo "miao" | tt ```

## Installation

```brew tap valentinaservile/topsy-turvy && brew install topsy-turvy```
